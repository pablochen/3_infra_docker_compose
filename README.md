# Vacation Program 

## 개발목표
- AS-IS : 데이터베이스만 도커라이즈
- TO-BE : 
  - 프론트엔드를 엔진엑스를 사용해 웹서버 구축 및 도커라이즈 
  - 백엔드 도커라이즈 
  - 하나의 도커컴포즈 파일로 실행

## 개발환경
- VueJs + npm
- Java8 +  SpringBoot
- MySql 

## 빌드 및 실행
- 실행 dir : /
  - 시작 : docker-compose up 
  - 종료 : docker-compose down
